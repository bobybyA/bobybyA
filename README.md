# Body by A - Personal Trainer Website

A modern, elegant website for online personal training coaching.

## Features

- Beautiful pastel gradient design (pink to purple)
- Responsive layout for all devices
- Smooth animations and interactions
- Contact form
- Service listings
- Client testimonials

## Setup

This is a static website that can be hosted on GitHub Pages, Netlify, or any static hosting service.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## GitHub Pages

This site is configured to work with GitHub Pages. Simply enable GitHub Pages in your repository settings and select the `main` branch as the source.
